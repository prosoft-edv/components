## Usage
Import the module into your module. 

```javascript
@NgModule({
  declarations: [MyComponent],
  imports: [PsSavebarModule]
})
export class MyModule {}
```

Now you can use it in your components like this:

```html
<ps-savebar>

</ps-savebar>
```
