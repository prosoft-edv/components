Escape:
    Use '' for escaping: '{0}'
    To escape a ' double it (like '')
    Examples:
        '{0}' -> {0}
        ''{0}'' -> 'value'
        '''{0}''' -> '{0}'
        This '{isn''t}' obvious -> This {isn't} obvious

    https://stackoverflow.com/a/32152170/2416833
